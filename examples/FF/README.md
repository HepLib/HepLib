### An Example in Fragmentation Funciton

- `cm` folder contains the items with the format `coefficint` $\times$ `integral F(ps,ns)`

- `N.cpp` is the program to perform numerical integration 

  ```
  heplib++ -o N N.cpp
  ./N -h # to show the options
  # to do numerical integration
  ./N -v 100 -z 13
  ```
